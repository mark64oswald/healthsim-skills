## Adding New Therapeutic Areas

When adding a new therapeutic area skill:

1. Create `{area}-trials.md` in this folder
2. Include YAML frontmatter with triggers
3. Document area-specific:
   - Common indications
   - Typical trial designs
   - Standard endpoints
   - Safety considerations
   - Regulatory pathways
4. Add at least 2 examples
5. Update this README
